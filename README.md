# Ass
# PRJ ASSIGNMENT | PetShop Web Store
This is a Web selling pet food and pet accessories.

# Main Features
User : Login, Signup, View Product, Add product to cart, Checkout
Admin: CRUD Product, Manage Order, CRUD Account

# Built With
- JavaOOP
- JSP/Severlet
- HTML/CSS
- Javascript
- SQLServer
